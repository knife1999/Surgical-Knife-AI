# 插件使用说明

## 页面总览

1. 单图处理：针对当前文档做一次 AI 生成与回贴。
2. 批处理：把多个任务加入队列后统一运行。
3. 与AI对话：选择模型后进行文字/图片对话。
4. 提示词查询：检索、收藏、使用、追加、删除已保存提示词。
5. 提示词新增：手动创建和维护提示词库条目。
6. 图片预览：上传多图，缩放、平移和轮播查看。
7. 使用说明：当前页面，帮助新用户快速上手。
8. 设置：集中管理大香蕉Key、AI对话 Key 和全局功能配置。

## 单图处理

- 配置 API 地址与 API Key 名称（来自设置页的大香蕉Key管理）。
- 填写提示词，设置分辨率、数量、超时、图层类型、压缩长边。
- 点击开始生成后，会对当前单图执行请求并回贴结果。
- 可将当前任务直接加入批处理队列。
- 支持查询额度与查看单次生成预览。
- 提示词区域支持快捷操作：清空提示词、保存当前提示词。

## 批处理

- 任务来源于单图页的“添加到批处理”。
- 队列中可查看文档名、参数摘要、提示词内容。
- 可逐条删除或整队清空，最后一键开始批量处理。
- 适合多文档、多提示词的连续生产场景。

## 与AI对话

- 在设置页先配置 AI对话 Key，再回到本页选择模型。
- 支持模型筛选、模型刷新时间查看。
- 对话支持文本与图片混合输入。
- 支持用户头像上传。
- 返回内容支持 Markdown 与代码块展示，代码支持一键复制。

## 提示词查询

- 支持按名称、分类、标签搜索。
- 支持只看收藏与从云端拉取提示词。
- 本地提示词支持删除；图书馆提示词只读。
- 每条提示词支持“使用”与“追加”到单图页输入框。

## 提示词新增

- 新增字段包含：名称、内容、描述、分类、标签。
- 保存后写入本地 JSON 存储。
- 可查看本地 JSON 路径与当前提示词总数。

## 图片预览

- 支持多图上传和轮播浏览。
- 支持滚轮缩放、拖拽平移、调整预览框高度。
- 适合在生成前后快速检查细节。

## 设置

- 大香蕉Key管理：新增、更新、删除、清空已存 Key。
- AI对话 Key管理：单独存储，不与大香蕉Key混用。
- 图层与压缩快捷设置：常用参数一键切换。
- 全局分区计算：对所有打开文档执行分区任务并输出汇总结果。

## 运行日志

- 页面底部统一输出运行过程与异常信息。
- 支持清空日志与隐藏日志面板。
- 建议问题排查时保留日志内容便于定位。

## 使用建议

1. 首次使用先到设置页完成大香蕉Key和AI对话 Key配置。
2. 单图验证通过后，再转批处理提高效率。
3. 常用提示词建议及时保存并分类，后续可快速复用。
4. 若接口异常，优先查看页面底部运行日志。
